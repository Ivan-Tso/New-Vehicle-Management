## 二、推荐方案详解

### 🥇 首选：Oracle Cloud Always Free（甲骨文云）

**为什么首选：** 配置最豪华，免费最彻底，国内能访问

规格（永久免费）：
- ARM服务器：4核 CPU + 24GB 内存
- 200GB 硬盘
- 每月10TB流量
- 完整Ubuntu虚拟机，root权限

优点：
- 永久免费（不是试用期，是Always Free）
- 配置远超需求（跑Flask绰绰有余）
- 选择东京/新加坡节点，国内访问延迟低
- 完整VM，随便装软件
- 不用Cloudflare，公司网络大概率不屏蔽

缺点：
- 注册较难（需要信用卡验证，可能多次失败）
- 需要Visa/Mastercard信用卡
- 可能需要上传身份证照片
- 账号审核可能等1-3天

注册步骤：
1. 打开 https://cloud.oracle.com/free/
2. 点"Start for Free"
3. 选择Home Region：**选东京(Tokyo)或新加坡(Singapore)**
   ⚠️ 区域选后不能改！
4. 填写姓名、邮箱、密码
5. 输入信用卡信息（Visa/Mastercard，扣1美元验证后退回）
6. 可能需要上传身份证/护照
7. 等待审核通过（1-3天）
8. 创建ARM实例：Ubuntu 22.04 + 配置SSH密钥

💡 注册技巧：
- 用Chrome无痕模式
- 信用卡姓名必须和注册姓名完全一致
- 如果失败，等48小时重试
- 很多人第2-3次才成功

---

### 🥈 备选：PythonAnywhere（最简单）

**为什么备选：** 注册最简单，2分钟上线，但配置有限

规格（永久免费）：
- 512MB 磁盘空间
- 每天100秒CPU时间
- 1个Web应用
- yourname.pythonanywhere.com 域名

优点：
- 只需邮箱注册，不需要信用卡
- 2分钟就能部署Flask应用
- 专门为Python Web设计
- 国内能访问（pythonanywhere.com没被墙）
- SQLite完美支持

缺点：
- 每天100秒CPU（够不够用？日常操作应该够，报表分析可能超）
- 只有512MB空间
- 没有自定义域名
- 应用闲久了会休眠（首次访问要等5-10秒唤醒）
- 每3个月必须登录一次，否则账号停用

注册步骤：
1. 打开 https://www.pythonanywhere.com/
2. 点"Create a free account"
3. 填邮箱和密码，完成
4. 上传Flask代码文件
5. 配置WSGI路径
6. 点Reload，上线！

### 🥉 第三选：Koyeb

规格（永久免费）：
- 0.1 vCPU / 256MB 内存
- 0.5GB 持久硬盘
- 50GB流量/月

优点：

缺点：

## 四、推荐路线

### 路线：先PythonAnywhere快速上线，同时申请Oracle Cloud

第一步（今天就能完成）：
→ 注册PythonAnywhere，部署系统

## 五、部署后的架构

```
公司电脑(浏览器) ──→ 云服务器(Flask+SQLite) ←── 手机(浏览器)
                        │
                        ├── 管理后台（电脑为主）
                        ├── 快速表单（手机为主）
                        └── 数据库 + CSV导出
```

电脑端：完整管理操作（录入、报表、导出）
手机端：简化的快速录入表单（行驶记录、费用记录）

## 六、手机端优化计划

部署到云服务器后，需要加：
1. 手机端简化表单页面（大按钮、少输入）
2. 行驶记录快速录入（默认今天日期、记住上次车辆）
3. 费用记录快速录入（常用费用类型快捷选择）
4. PWA支持（添加到手机主屏幕，像App一样打开）
